Tipos de Target

Para certas ocasiões para melhorar a experiencia do usuário e a visibilidade do seu site é usado TARGET nos link para ocasiões como:

Leva o usuário para uma nova janela em branco _blank

Vai abrir o link no frame atual _self

Vai desfazer todos os frames e abrir o destino no navegador completo _top

Similar ao uso do _top em uma referência à janela mãe _parent

Caso esteja usando frames, indicar o nome da janela à abrir nome-do-frame

Atributo rel

Serve pra indicar a natureza do link

1.0 next indica o link é para a próxima parte do documento atual

2.0 prev Indica que o link é para a parte anterior do documento atual

3.0 author indica que é um link parao site do autor do artigo atual

4.0 external indica que é um link para outro site que não faz parte do site

5,0 nofollow indica que é um link para um site não endossado, como um link pago
